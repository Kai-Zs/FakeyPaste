## 📖 介绍

**FakeyPaste** 不是剪贴板管理器，而是一款智能的模拟键盘输入工具。它通过模拟真实的键盘按键操作，将文本逐字符输入到目标输入框中，完美解决网页表单、受限系统等场景下无法直接粘贴的问题。

---

## ✨ 功能特点

- 🖥️ **模拟真实键盘输入** - 逐字符模拟按键，绕过粘贴限制
- ⚡ **快捷键支持** - 全局快捷键控制，无需切换窗口
- ⏸️ **暂停/恢复** - 输入过程中可随时暂停和恢复
- ⚙️ **自定义延时** - 可调节字符间隔和开始延时
- 📋 **剪贴板集成** - 一键从剪贴板导入内容
- 🧹 **缩进清理** - 支持清空代码缩进（特别适合阿尔法平台）
- 🎨 **精美界面** - 现代化深色主题设计

## 📝 使用说明

1. **载入内容**
   - 点击「从剪贴板粘贴」将代码载入预览区
   - 或直接在预览区输入文本

2. **准备输入**
   - 切换到目标窗口，将光标定位到输入位置

3. **开始输入**
   - 点击「开始输入」按钮
   - 或使用快捷键 `Ctrl+Shift+P`
   - 在倒计时结束前切换到目标窗口

4. **控制输入**
   - 暂停：点击「暂停输入」或按 `Ctrl+Shift+L`
   - 继续：点击「继续输入」或按 `Ctrl+Shift+P`
   - 停止：点击「停止输入」

---

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+P` | 开始输入 / 恢复输入 |
| `Ctrl+Shift+L` | 暂停输入 |

---

## ⚙️ 配置选项

- **字符间隔**：每个字符输入的间隔时间（毫秒），默认 50ms
- **开始延时**：点击开始后延迟多少秒开始输入，默认 3秒
- **继续延时**：暂停后恢复输入的延迟时间，默认 3秒

---

## 💡 (阿尔法平台特别提示)

在阿尔法平台上使用时，复制代码后点击「清空缩进」按钮，因为平台 IDE 会自动添加缩进，避免重复缩进导致代码格式错误。
